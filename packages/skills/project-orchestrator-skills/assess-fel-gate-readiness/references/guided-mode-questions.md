# Guided Mode Questions — Assess FEL Gate Readiness

## Mode Selection

> "Para evaluar la readiness del gate FEL necesito entender el alcance y estado del proyecto.
> Selecciona el modo de trabajo:
>
> 1. **Proceso guiado completo** — Te guío paso a paso por toda la información necesaria.
> 2. **Solo contexto inicial** — Recopilo lo mínimo imprescindible y ejecuto con supuestos.
> 3. **Cargar documentos directamente** — Ya tienes los documentos listos."

## GM-1: Input Discovery

| # | Question | Maps to Input | Mode | Acceptable Answers |
|---|----------|--------------|------|-------------------|
| 1 | "¿Cuál gate FEL se está evaluando? (FEL1 — Identificación/Prefactibilidad, FEL2 — Selección/Factibilidad, FEL3 — Definición/FEED)" | Target FEL Gate | ESSENTIAL | FEL1, FEL2, or FEL3. Determines which criteria apply. |
| 2 | "Describe el alcance del proyecto: nombre, tecnología, capacidad, ubicación. ¿Ha cambiado el alcance desde el gate anterior?" | Project Scope | ESSENTIAL | Scope description. If FEL2/3: note any scope changes since prior gate. |
| 3 | "¿Cuáles workstreams están en el alcance del assessment? (ingeniería, construcción, operaciones, HSE, comercial, ambiental, financiero)" | Workstreams in Scope | ESSENTIAL | List of workstreams. Default: all. |
| 4 | "Para cada workstream, describe el estado actual: % completado, entregables clave producidos, issues abiertos." | Workstream Status | ESSENTIAL | Status by workstream. At minimum: % complete and key deliverables. |
| 5 | "¿Cuáles son los entregables clave requeridos para este gate? Comparte la lista o checklist si existe." | Gate Deliverable Inventory | ESSENTIAL | Deliverable list or "usar checklist estándar IPA/VSC". |
| 6 | "¿Se pasó el gate anterior? Si es FEL2 o FEL3, comparte el resultado del gate previo y las condiciones/acciones pendientes." | Previous Gate Results | OPTIONAL | Results + pending actions. N/A for FEL1. |
| 7 | "¿Quiénes son los decision-makers del gate? (comité de inversiones, sponsor, board)" | Gate Decision Makers | OPTIONAL | Names and roles. |
| 8 | "¿Cuál es la fecha target para el gate review? ¿Es flexible?" | Gate Date | OPTIONAL | Date or "por definir". |
| 9 | "¿Hay condicionantes o deal-breakers conocidos que podrían bloquear el gate?" | Known Blockers | OPTIONAL | List or "ninguno identificado". |
| 10 | "¿Tienen estimación de costos actualizada (Class de AACE)? ¿Cuál es el accuracy range?" | Cost Estimate Status | OPTIONAL | AACE class (1-5) + range, or "en desarrollo". |
| 11 | "¿Tienen modelo financiero/business case actualizado?" | Business Case Status | OPTIONAL | Yes/No + key metrics (NPV, IRR) or "en desarrollo". |

## Dependency Checks

- [ ] For FEL2: Was FEL1 gate passed? → Check for pending conditions from FEL1
- [ ] For FEL3: Was FEL2 gate passed? → Check for pending conditions from FEL2
- [ ] OR Strategy available? → Provides context for OR workstream assessment
- [ ] Risk register updated? → Required for risk-related gate criteria
