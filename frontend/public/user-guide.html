Section 04

Work Requests

Create a Work Request

  1. Click on + New WR (green button)
  2. Fill in the fields: Equipment (search by TAG or name), Failure Description, Priority and Production Impact
  3. AI automatically classifies the failure, suggests priority, detects duplicates and generates a descriptive title
  4. Click Submit

Priorities

P1 Immediate P2 High P3 Medium P4 Low

Manage Work Requests

  • Filters: By status, priority, area, equipment
  • Approve Validate and approve the request
  • Reject Reject with reason
  • Create WO Convert to Work Order
  • Cancel WR Cancel the request (with confirmation)
  • Close Close without action
Duplicate Detection: When creating a WR, the system shows similar WRs in a side-by-side carousel format.

WR Status Flow

DRAFT → PENDING → VALIDATED → APPROVED → IN_PROGRESS → COMPLETED → CLOSED ↘ REJECTED ↘ CANCELLED ↘ DELETED

Deleted (Recycle Bin)

  • "Deleted" tab to view deleted WRs
  • Only admin/manager can Restore or Permanently Delete
  • Deletion reason is recorded

Planning

Create Work Order from WR

  1. In Work Requests, select an approved WR
  2. Click Create WO
  3. Automatically generated with WO number (e.g.: WO-2026-00042), WR data and planned dates

Plan Work Order

  1. Select a WO in CREATED status
  2. Fill in: Operations (type, specialty, hours), Materials (SAP search), Work Center, Dates and Budget
  3. Click Plan to change status to PLANNED

External Vendor

  • Click + External Service to add a vendor

Section 14

Data Import

Import Process

  1. Upload: Drag or select Excel (.xlsx) or CSV (.csv) file — Max 50MB
  2. Preview: View first rows and columns. For multi-sheet Excel, select the sheet
  3. Column Mapping: Automatic (detects by name) or Manual (drag columns)
  4. Target Table: Select destination table (equipment, WRs, WOs, etc.)
  5. Import Mode: Append (add new), Replace (replace all), or Merge (update + add)
  6. Execute: Click Import to run
  7. History: Review import history
AI-Assisted Import: Enable the "AI Detection" toggle for AI to suggest optimal target table, detect data quality issues and map columns automatically.

Section 18

Frequently Asked Questions

I can't see a module in the sidebar
Your role doesn't have permissions. Contact your administrator.
KPIs show "—"
Not enough data to calculate. Import historical data or wait for records to accumulate.
How do I change my password?
Settings > Profile > Change Password
How do I export data?
Reports > select template > Download (XLSX/CSV). Or use Data Export on the same page.
The report doesn't filter by my plant
Make sure you have a plant selected in the top selector.
How do I install the app on my phone?
Open the URL in Chrome/Safari > Menu > "Add to Home Screen" / "Install App"
Can I use the app without internet?
Yes, the app automatically caches data. On reconnection, it updates automatically.
Where do I report a bug?
Use the Feedback button (bottom left corner) to report issues.
How are KPIs calculated?
MTBF = Total operating hours / Number of failures
MTTR = Total repair hours / Number of repairs
Availability = MTBF / (MTBF + MTTR) x 100
OEE = Availability x Performance x Quality
