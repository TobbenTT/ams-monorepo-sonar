## Templates & References

### Templates (stored in `methodology/templates/commissioning/`)

1. **TCCC Certificate Template (.docx)** — comprehensive certificate template as defined in Output Specification
   - Pre-formatted with all 10 sections
   - Signature block with 4 mandatory + 1 optional signatories
   - Appendix templates for equipment list, procedure list, punchlist extract
   - Instructions for completion embedded in template (hidden text)

2. **System Readiness Matrix Template (.xlsx)** — standard 6-sheet workbook as defined in Output Specification
   - Pre-built conditional formatting with traffic light rules
   - Pivot tables and dashboard charts
   - System breakdown structure auto-populated from project master
   - Critical path analysis formulas

3. **PSSR Checklist Template** — comprehensive pre-startup safety review checklist
   - Process safety items (relief devices, safety interlocks, SIS logic)
   - Equipment safety items (guards, grounding, emergency stops, lighting)
   - Operational safety items (procedures, training, emergency equipment, PPE)
   - Environmental safety items (emissions controls, spill containment, waste management)
   - Regulatory compliance items (permits, licenses, notifications)

4. **Post-RFSU Monitoring Log Template** — structured performance tracking
   - Daily KPI recording (availability, throughput, quality, safety)
   - Weekly summary with trend analysis
   - Issues log with classification and resolution tracking
   - 30/60/90-day review assessment template
   - Final Acceptance recommendation template
